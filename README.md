# Personal Financial Statistics

This project is an Angular application designed to help users manage and analyze their personal financial statistics.

## Features

- User-friendly interface for tracking expenses and income.
- Visual representation of financial data through charts and graphs.
- Environment configurations for development and production.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/yourusername/personal-financial-statistics.git
   ```

2. **Navigate to the project directory:**
   ```
   cd personal-financial-statistics
   ```

3. **Install dependencies:**
   ```
   npm install
   ```

4. **Run the application:**
   ```
   ng serve
   ```

5. **Open your browser and navigate to:**
   ```
   http://localhost:4200
   ```

## Project Structure

- `src/app`: Contains the main application components.
- `src/assets`: Static assets such as images and fonts.
- `src/environments`: Environment-specific configurations.
- `angular.json`: Angular CLI configuration.
- `package.json`: Project dependencies and scripts.
- `tsconfig.json`: TypeScript configuration.
